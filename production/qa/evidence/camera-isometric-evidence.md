# Camera Isometric Evidence

**Date**: 2026-04-23  
**Scope**: `CM-001`

## Current State

- Original 2.5D/isometric direction caused interaction/readability problems in user testing:
  - units were hard to select
  - camera felt awkward
  - terrain height was hard to read
- Formal battle scene was therefore switched to a 2D top-down readable grid for the vertical slice.
