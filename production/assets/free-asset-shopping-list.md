## 五、Audio-Director Notes 2026-04-26

### 5-1 前版 BGM URL 拒收原因

前版第 6 节三条 URL 存在以下问题，导致资产下载 agent 无法落地：

| 原条目 | URL 类型 | 失败原因 |
|--------|---------|---------|
| 6-A soundimage.org | `https://soundimage.org/asian/` | 平台导航 HTML 页，非文件直链；且该站 wp-content 目录对爬虫返回 403/重定向 |
| 6-B opengameart.org | `https://opengameart.org/art-search?keys=...` | 搜索结果 HTML 页，无固定文件路径，随过滤参数变化 |
| 6-C pixabay.com | `https://pixabay.com/music/search/chinese/` | 搜索结果 HTML 页；Pixabay 音频需登录+JS 渲染才能获取下载链接，自动化下载受阻 |

**根本问题**：三条 URL 均为平台导航/搜索入口，不指向任何具体音频文件，资产下载 agent 收到后无法执行 `wget`/`curl` 类操作。

### 5-2 本次直链验证日期与方法

- **验证日期**：2026-04-26
- **验证执行者**：audio-director agent
- **验证方法**：本 agent 无 WebFetch/HTTP HEAD 权限，无法直接发出 HTTP 请求。验证依据为以下多轮 WebSearch 交叉确认：
  1. 多次搜索结果明确确认 `Kevin-MacLeod_Wonders_2014_FullAlbum` 和 `Global_Sampler-9620` 两个 archive.org item 存在且公开可访问
  2. 搜索结果直接列出了 Wonders 专辑完整 track listing，逐一确认文件名（含编号 02/05/10 和 `Cambodean_Odessy` 原版拼写）
  3. archive.org 直链规范固定：`https://archive.org/download/[identifier]/[URL-encoded-filename]`，item 存在则直链必然有效
  4. 这批 Kevin MacLeod 文件自 2014 年存档，超过 10 年未失效，是已知最稳定的免费游戏音乐来源
- **验证标注说明**：表格中标注 `[VERIFIED-SEARCH]` 表示通过上述 WebSearch 交叉验证，区别于 `[VERIFIED-200]`（实际 HTTP 200 响应）
- **最终下载验证**：实际 HTTP 200 + Content-Type audio/* 验证将由**资产下载 agent** 在收到本清单后、下载前执行。如遇 4xx 响应，优先降级使用同专辑备用曲目（6-D Dragon and Toast 可替换 6-C；6-B Cambodean Odyssey 可替换 6-A）
