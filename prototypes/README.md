# SRPG Vertical Slice Prototype

> 版本: v0.1 | 日期: 2026-04-20 | 状态: 规划中

## 概述

本原型用于验证SRPG核心玩法循环的可行性，包括：
- 回合制战斗（速度序列制）
- 基础属性系统
- 职业选择
- 装备系统

## 目标

在Pre-Production阶段创建一个可玩的垂直切片，包含：
1. 完整的战斗循环（开始→行动→结束）
2. 基础角色养成
3. UI/HUD展示
4. 存档功能

## 核心系统优先级

1. 战斗系统（回合制 + 速度序列）
2. 基础属性
3. 职业系统（基础职业）
4. 装备系统（基础）

## 验证标准

- [ ] 战斗可正常进行
- [ ] 伤害计算正确
- [ ] 回合顺序正确
- [ ] 角色可装备道具
- [ ] 可保存/读取游戏

## 文件结构

```
prototypes/
├── combat-prototype/     # 战斗系统原型
├── attribute-prototype/  # 属性系统原型
└── vertical-slice/      # 垂直切片
```
