信念值三维度：仁（ren）/ 义（yi）/ 智（zhi），累计值上限 100，跨战斗持久化。

**Ch.2 第一次分叉阈值（Ch.2-1 结算后）：**
```
if (yi - max(ren, zhi)) >= 5 → 走义路线（chapter_02_suppression，2B）
else → 走仁/智路线（chapter_02_escort_mercy，2A）
```

阈值 5 是有意设计的低门槛，确保 Ch.1 中小幅偏义的玩家也会感知到分叉。
均衡玩家默认走 2A（护送战，教学性更强）。

**Why:** 总纲要求"信念值让分支感觉自然生长而非二选一"，阈值 5 保证分叉足够灵敏，
同时不因 Ch.1 的微小随机偏差把玩家推进错误路线。

**How to apply:** 写 Ch.3+ GDD 时，需在 Ch.2-3 结算后验证三路线差值 ≥ 15，
确保 Ch.3 能用更高阈值做更深分叉。后续分叉阈值应逐章递增（建议 Ch.3 阈值 ≥ 15）。

**已规划的 Ch.2 信念值触发点：**
- Ch.2-1 开场叙事选择：驱散流民(仁-5/义+8/智+3) vs 收留流民(仁+10/义-3/智+5)
- Ch.2-2A 护送结果：NPC 存活(仁+12)，NPC 阵亡(仁-8)
- Ch.2-2B 结算：豪强 NPC 击杀数超我方(义+10/智-5)
- Ch.2-3 开场对话：喊话劝降(仁+6/义-3) vs 直接接战(义+6/仁-6)
