## 1. Screen Overview

暂停菜单在战斗或探索中按 Esc/Start 触发，冻结游戏时间并以卷轴覆盖层显示。采用屏幕空间叠加（场景暂停，遮挡不影响体验——art-bible 7.1.3）。

**玩家目标**: 快速恢复游戏，或进行存档/读档/调整设置/退出。

## 3. Component Inventory

### 3.1 背景遮罩 (Backdrop)

| 属性 | 值 |
|------|-----|
| 覆盖 | 全屏 |
| 效果 | 场景模糊（景深 blur）+ 浓墨半透明覆盖 40% |
| 动画 | 0.3s 渐入 |

### 3.2 暂停面板 (Pause Panel)

| 属性 | 值 |
|------|-----|
| 位置 | 屏幕右侧居中 |
| 渲染 | 屏幕空间叠加 |
| 尺寸 | 320×480px (1080p) |
| 材质 | 宣纸底色 #F5F2EB + 不规则毛边 + 5-10% 墨渍 |
| 出场动画 | 0.3s 卷轴从右向左展开 |

**选项列表**:

| 序号 | 选项 | 条件 | 目标 |
|------|------|------|------|
| 1 | 继续 | 始终 | 关闭暂停菜单 |
| 2 | 存档 | 非战斗状态可存 | 存档槽选择 |
| 3 | 读档 | 始终 | 存档槽选择 |
| 4 | 设置 | 始终 | 设置子面板 |
| 5 | 返回标题 | 始终 | 二次确认 |
| 6 | 退出游戏 | 始终 | 二次确认 |

**选项交互状态**: 与 main-menu.md 一致（金铜色焦点 + 金黄悬停）。

### 3.3 存档/读档子面板 (Save/Load Sub-Panel)

替换暂停面板内容，不新开覆盖层：

```
┌────────────────────────────┐
│  存档 / 读档      [返回]   │
├────────────────────────────┤
│                             │
│  ┌────────────────────────┐│
│  │ 槽位 1  [主角] Lv.25  ││
│  │ 第三章  2026-04-23     ││
│  └────────────────────────┘│
│                             │
│  ┌────────────────────────┐│
│  │ 槽位 2  [主角] Lv.8   ││
│  │ 第一章  2026-04-22     ││
│  └────────────────────────┘│
│                             │
│  ┌ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐│
│  │ 槽位 3  空             ││
│  └ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┘│
│                             │
│  ┌────────────────────────┐│
│  │ ... 槽位 4-8          ││
│  └────────────────────────┘│
│                             │
└────────────────────────────┘
```

**存档 vs 读档差异**:

| 场景 | 默认焦点 | 空槽位 | 覆盖确认 |
|------|----------|--------|----------|
| 存档 | 最近存档槽 / 第一个空槽 | 可选（虚线框） | 覆盖已有存档时弹出二次确认 |
| 读档 | 最近存档槽 | 灰化不可选 | 无 |

**存档覆盖确认**:

```
┌─────────────────────────────────┐
│ ⚠ 确认覆盖                      │
│                                 │
│ 该槽位已有存档，是否覆盖？       │
│ 旧存档将永久丢失。              │
│                                 │
│    [取消]        [覆盖]         │
└─────────────────────────────────┘
```

## 4. Screen States & Transitions

| 过渡 | 时长 | 视觉 |
|------|------|------|
| 打开暂停 | 0.3s | 场景 blur + 浓墨遮罩渐入 + 面板卷轴展开 |
| 关闭暂停 | 0.3s | 面板收起 + 遮罩渐出 + 场景清晰化 |
| 切换子面板 | 0.2s | 内容交叉淡入淡出 |
| 确认对话框 | 0.15s | 对话框从中心放大出现 |

**状态栈**:

```
暂停主菜单
  ├── 存档面板
  │     └── 覆盖确认
  ├── 读档面板
  ├── 设置面板
  │     └── 按键重映射
  ├── 返回标题确认
  └── 退出确认
```

Esc 始终弹出栈顶一级。

## 5. Input Mapping

### 5.1 键盘/鼠标

| 操作 | 输入 |
|------|------|
| 上下选择 | ↑/↓ / 鼠标悬停 |
| 确认 | Enter / 左键 |
| 返回/取消 | Esc / 右键 |
| 关闭暂停菜单 | Esc（在主层级时） |

### 5.2 游戏手柄

| 操作 | 输入 |
|------|------|
| 上下选择 | 方向键 / 左摇杆 |
| 确认 | A |
| 返回/取消 | B |
| 关闭暂停菜单 | B / Start（在主层级时） |

## 6. Contextual Rules

| 游戏状态 | 暂停行为 |
|----------|----------|
| 探索 | "存档"可用 |
| 战斗中（玩家回合） | "存档"禁用，灰色显示 "战斗中不可存档" |
| 战斗中（敌方回合） | 同上 |
| 过场演出 | 暂停不可用，Esc 跳过过场 |
| 自动战斗 | 暂停时自动战斗挂起，恢复后继续 |

---

## 7. Accessibility

| 要求 | 实现方式 |
|------|----------|
| 焦点可见 | 金铜色边框 2px |
| 键盘全导航 | Esc/Enter/方向键完整操作 |
| 设置持久化 | 所有设置立即生效并保存 |
| 确认保护 | 所有破坏性操作（覆盖存档/退出）需二次确认 |
| 文字缩放 | 暂停菜单文字跟随全局文字大小设置 |
