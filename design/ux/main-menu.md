## 1. Screen Overview

主菜单是玩家启动游戏后的首个界面。采用水墨画框 + 卷轴展开风格，背景为 3D 水墨山水场景（缓慢飘动的粒子+光影），营造"可游历山水长卷"的第一印象。

**玩家目标**: 快速进入游戏（继续/新开），或调整设置。

## 3. Component Inventory

### 3.1 标题区 (Title Area)

| 属性 | 值 |
|------|-----|
| 位置 | 屏幕上方 1/3 居中 |
| 渲染 | 屏幕空间叠加 |
| 动画 | 0.8s 卷轴展开出现（art-bible 7.4.3） |

| 组件 | 规格 |
|------|------|
| 游戏标题 | 仿古印刷体 48-64px（1080p），焦墨 #1A1A1A |
| 副标题 | 楷书 18px，淡墨 #8A8A8A |
| 装饰 | 两侧水墨山水纹样 10-15% 透明度 |
| 底框 | 宣纸质感 #F5F2EB + 2px 淡墨边框 |

### 3.2 主菜单选项 (Menu Options)

| 属性 | 值 |
|------|-----|
| 位置 | 屏幕中央偏右 |
| 渲染 | 屏幕空间叠加，卷轴容器 |
| 容器 | 宣纸质感背景 + 不规则毛边 + 5-10% 墨渍 |

**选项列表**:

| 序号 | 选项 | 条件 | 目标 |
|------|------|------|------|
| 1 | 继续游戏 | 有存档时显示 | 存档槽选择 |
| 2 | 新游戏 | 始终显示 | 确认 → 世界观开场 |
| 3 | 设置 | 始终显示 | 设置界面 |
| 4 | 图鉴 | 有解锁内容时显示 | 图鉴界面 |
| 5 | 退出 | 始终显示 | 退出确认 |

**选项交互状态**:

| 状态 | 视觉 |
|------|------|
| 默认 | 宣纸底色，焦墨文字，居中对齐 |
| 悬停/焦点 | 金铜色边框 #B8860B 2px + 文字变金黄 #D4A843 + 轻微放大 102% |
| 选中 | 金铜色脉冲动画 + 确认音 |
| 禁用 | 灰墨文字 #BEBEBE + 半透明 60% |

## 4. Screen States & Transitions

### 4.1 进入主菜单

| 序列 | 时长 | 视觉 |
|------|------|------|
| 1. 背景加载 | 0.5s | 3D 水墨山水场景淡入 |
| 2. 标题展开 | 0.8s | 卷轴从中心向上下展开 |
| 3. 菜单选项出现 | 0.4s | 选项依次从上到下淡入（间隔 0.05s） |
| 4. 焦点锁定 | — | 默认焦点在"继续游戏"（有存档）或"新游戏" |

### 4.2 离开主菜单

| 目标 | 过渡 |
|------|------|
| 进入游戏（继续/新游戏） | 菜单卷轴收起 0.3s → 场景淡出 0.5s → 游戏加载 |
| 进入设置 | 设置面板从右滑入 0.3s |
| 进入图鉴 | 图鉴面板从右滑入 0.3s |

### 4.3 首次启动 vs 再次启动

| 状态 | 差异 |
|------|------|
| 首次启动（无存档） | "继续游戏"隐藏，默认焦点"新游戏"，标题动画播放完整 |
| 再次启动（有存档） | "继续游戏"为默认焦点，标题动画可跳过（任意键） |

## 5. Input Mapping

### 5.1 键盘/鼠标

| 操作 | 输入 |
|------|------|
| 上下选择 | ↑/↓ 方向键 / 鼠标悬停 |
| 确认 | Enter / 左键 |
| 返回/取消 | Esc / 右键 |
| 跳过标题动画 | 任意键 |

### 5.2 游戏手柄

| 操作 | 输入 |
|------|------|
| 上下选择 | 方向键 / 左摇杆 |
| 确认 | A |
| 返回/取消 | B |
| 跳过标题动画 | 任意键 |

## 6. Accessibility

| 要求 | 实现方式 |
|------|----------|
| 焦点可见 | 金铜色边框 2px + 文字变色 |
| 键盘全导航 | 方向键 + Enter/Esc 完整导航 |
| 选项可读性 | 选项间距 ≥ 48px（1080p），字号 ≥ 18px |
| 取消操作 | Esc 始终可返回上一级 |
| 退出确认 | 二次确认，防止误触 |
