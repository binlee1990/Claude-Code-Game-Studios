# Interaction Pattern Library

> 版本: v0.1 | 日期: 2026-04-20 | 状态: 初始化

---

## 1. 概述

本文档记录游戏中使用的标准交互模式，确保UI/UX一致性。

---

## 2. 导航模式

### 2.1 菜单导航

**键盘**: Tab跳转, 方向键选择, Enter确认, ESC取消
**手柄**: 方向键选择, A确认, B取消, LB/RB翻页
**触摸**: 点击选择

### 2.2 战斗目标选择

**网格高亮**: 悬停单元格显示青绿色高亮
**选中状态**: 选中目标显示白色描边
**范围预览**: 可攻击范围显示深红色半透明覆盖

## 4. 确认模式

### 4.1 二次确认对话框

```
┌─────────────────────────────────┐
│ ⚠ 确认操作                      │
│                                 │
│ 是否确认 [操作描述]？            │
│                                 │
│    [取消]        [确认]         │
└─────────────────────────────────┘
```

### 4.2 危险操作确认

红色边框 + 警告图标 + 5秒倒计时取消

---

## 5. 反馈模式

### 5.1 操作成功

- 绿色粒子爆发
- 音效: 上升音阶
- 持续时间: 0.5秒

### 5.2 操作失败

- 红色闪烁
- 音效: 低沉警告音
- 震动: 100ms手柄震动

### 5.3 加载状态

- 圆形进度指示器
- 当前状态文字描述
- 可取消（如果适用）

---

## 6. 表单模式

### 6.1 输入字段

- 默认状态: 1px #4A9EFF边框
- 焦点状态: 2px #4A9EFF边框
- 错误状态: 2px #FF4A4A边框 + 错误提示文字

### 6.2 下拉选择

- 展开动画: 200ms ease-out
- 选项悬停: 背景色#4A9EFF 20%透明

---

## 7. 拖放模式

### 7.1 物品拖放

- 拖动中: 物品50%透明，跟随光标
- 有效目标: 绿色高亮边框
- 无效目标: 红色边框，物品返回原位
