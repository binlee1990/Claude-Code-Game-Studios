# 世界观/叙事系统

> **Status**: Designed (pending review)
> **Author**: binlee1990 + agents
> **Last Updated**: 2026-04-22
> **Implements Pillar**: 信念值驱动三路线分支叙事——系统互锁的情感层面

## Overview

世界观/叙事系统是 SRPG 的情感基础设施，定义游戏世界的背景设定、势力格局、角色体系，以及驱动三路线分支叙事的信念值机制。该系统同时承担两个角色：作为**叙事层**，玩家在每一场战斗、每一次选择中直接体验故事；作为**Foundation 层**，为属性系统（五行元素的文化根基）、羁绊系统（角色关系网络）、美术风格（水墨山水美学）提供世界观一致性保障。

世界观采用**架空中国历史**框架——以真实历史为骨架，加入虚构势力、人物和事件。玩家熟悉的历史框架降低理解门槛，架空元素提供叙事自由度。可融合多个朝代元素，不受史实束缚。

叙事结构采用**信念值驱动三路线分支**：玩家行为累积影响立场（不只是剧情选择点），在关键节点根据累积值自然分流。每条路线有独立故事、独立角色、不同道德代价。

核心设计原则：**选择=失去**——每个选择都意味着失去其他可能性，让决策具有真正的重量。

## Player Fantasy

- **历史参与者幻想**：玩家不是旁观者，而是站在历史十字路口的决策者。每一次战斗、每一次对话、每一次资源分配都在回答同一个问题——"我相信什么？" 玩家感受到自己正在书写一段属于这个架空世界的历史。
- **选择重量的幻想**：信念值系统让选择感觉自然生长而非突兀的二选一。当分流终于发生时，玩家回望整个旅程会说"我确实做出了这个选择"而非"游戏把我推到这里"。
- **失去的幻想**：三路线意味着玩家永远无法一次体验全部内容。失去的角色、未走的路线、不可逆的牺牲——这些"失去"是叙事的情感锚点，也是多周目的核心驱动力。
- **参考体验**：皇骑L/N/C路线的道德复杂性（没有正确答案，只有立场选择）+ 三角战略的信念值累积感（小选择堆叠成大分流）+ 魂系叙事的留白与推测空间。

## Detailed Design

### Core Rules

**规则 1 — 世界观框架**

游戏世界基于**架空中国历史**，融合多个朝代元素，不受单一史实束缚。

| 世界观要素 | 设计原则 | 说明 |
|------------|----------|------|
| 历史骨架 | 真实历史为底色 | 玩家熟悉的政治结构（朝廷/诸侯/战争）降低理解门槛 |
| 架空元素 | 虚构势力、人物、事件 | 提供叙事自由度，不受史实约束 |
| 政治基调 | 政治阴谋 + 道德灰色地带 | 无绝对善恶，每个势力有合理动机（皇骑式成熟叙事） |
| 角色道德 | 非脸谱化 | 每个角色有复杂动机，盟友可能是敌人，敌人可能有理 |
| 元素体系 | 中国五行（火/水/风/土/金） | 世界观层面为五行；战斗层面仅火/水/风/土四元素参与克制（金为纯叙事元素，不参与战斗） |

**势力格局**（初步框架，具体设定待细化）：

| 势力类型 | 数量 | 叙事功能 |
|----------|------|----------|
| 朝廷/中央 | 1 | 秩序、传统、保守——代表"墨黑路线"的价值观 |
| 叛军/改革 | 1 | 变革、激进、行动——代表"朱红路线"的价值观 |
| 隐世/中立 | 1 | 平衡、探索、调和——代表"青绿路线"的价值观 |
| 其他势力 | 2-3 | 填充世界观深度，提供多维度冲突 |

**规则 2 — 信念值系统**

信念值是驱动三路线分流的数值机制。玩家在游戏过程中的所有行为累积影响三个信念维度：

| 信念维度 | 对应路线 | 积累行为 | 色彩标识 |
|----------|----------|----------|----------|
| **果断** (Resolve) | 朱红路线 | 进攻性战斗选择、直接对话、牺牲少数救多数 | #D4501A |
| **隐忍** (Patience) | 墨黑路线 | 防御性战斗选择、暗中行动、忍辱负重 | #1E2A3A |
| **探索** (Curiosity) | 青绿路线 | 探索隐藏区域、支线任务、帮助所有人 | #3D8B5F |

**信念值来源**：

| 来源 | 权重 | 示例 |
|------|------|------|
| 对话选择 | 高 | 在对话中选择"立即行动"→ +3 果断 |
| 战斗行为 | 中 | 主动攻击弱点 → +1 果断；防守反击 → +1 隐忍 |
| 任务完成方式 | 高 | 潜入完成任务 → +3 隐忍；正面突破 → +3 果断 |
| 羁绊互动 | 中 | 与某路线角色加深羁绊 → 对应维度 +1 |
| 资源分配 | 低 | 将资源投入探索 → +1 探索 |

**规则 3 — 三路线分支机制**

游戏包含 3-5 个关键分支点。在每个分支点，系统检查累积信念值，决定分流方向。

```
分流判定：
  IF max(果断, 隐忍, 探索) > second_max + threshold:
      → 进入对应路线
  ELSE:
      → 提供选择（两个最高维度的路线可选）
```

| 分支点 | 章节范围 | 类型 | 影响 |
|--------|----------|------|------|
| 分支 1（教程后） | 第 2 章 | 软分支 | 影响可用角色和任务方向，不锁定路线 |
| 分支 2（中期） | 第 5 章 | 硬分支 | 锁定路线，不可逆转 |
| 分支 3（后期） | 第 8 章 | 深度分支 | 路线内再分支，影响结局 |
| 分支 4（最终） | 第 10 章 | 结局分支 | 决定最终结局 |
| 分支 5（隐藏） | 多周目 | 特殊分支 | 条件触发，二周目+特定信念组合解锁 |

**路线差异**：

| 维度 | 朱红路线 | 墨黑路线 | 青绿路线 |
|------|----------|----------|----------|
| 叙事主题 | 革命的代价 | 秩序的维护 | 真相的追寻 |
| 可用角色 | 叛军阵营角色 | 朝廷阵营角色 | 中立/隐世角色 |
| 独有 Boss | 镇压军将领 | 叛军首领 | 远古守护者 |
| 道德困境 | 牺牲无辜换胜利 | 压迫弱者保秩序 | 优柔寡断致混乱 |
| 独有装备 | 烈焰系列 | 暗影系列 | 自然系列 |
| 结局类型 | 激进的胜利 | 苦涩的和平 | 开放的真相 |

**规则 4 — 角色叙事规则**

| 规则 | 说明 |
|------|------|
| 预设主角 | 固定姓名/背景/动机，玩家通过选择塑造性格 |
| 角色退场 | 战败→重伤退场→专属退场剧情→可通过任务招回 |
| 羁绊影响叙事 | 与某路线角色达到 S 级羁绊 → 该路线分支点获得额外选项 |
| 角色冲突 | 不同路线的角色之间存在不可调和的矛盾，选择路线意味着失去部分角色 |
| NPC 记忆 | NPC 记住玩家的行为，后续对话反映之前的选择 |

**规则 5 — 叙事内容结构**

| 内容层级 | 数量（估算） | 说明 |
|----------|-------------|------|
| 主线章节 | 10-12 章 | 核心叙事，所有路线共享前期 |
| 路线专属章节 | 3-5 章/路线 | 分支后每条路线的独立内容 |
| 支线任务 | 20-30 个 | 世界观深度，信念值积累来源 |
| 羁绊事件 | 每角色 4-6 个 | 角色发展 + 羁绊等级提升 |
| 世界观碎片 | 50+ 条 | 物品描述/环境叙事/隐藏文本 |

### States and Transitions

**叙事状态机**：

```
Prologue → Chapter_Play → Branch_Point → Route_Chapter → Finale
               ↑               ↓
               └── Side_Quest ──┘
```

| 状态 | 行为 | 允许的操作 |
|------|------|------------|
| **Prologue** | 强制线性叙事 | 只能推进 |
| **Chapter_Play** | 主线推进 + 可选支线 | 接受任务、战斗、对话、探索 |
| **Side_Quest** | 支线任务执行 | 接受/完成/放弃支线 |
| **Branch_Point** | 信念值判定 + 确认 | 确认选择（软分支可能跳过确认） |
| **Route_Chapter** | 路线专属内容 | 路线内推进 |
| **Finale** | 结局 | 触发结局动画 + 成就结算 |

**信念值累积状态**：

```
Early_Game (信念值 < 30%) → 双维度可选范围大
Mid_Game (信念值 30-70%) → 主导维度开始明显
Late_Game (信念值 > 70%) → 分流方向确定
```

| 系统 | 交互类型 | 数据流向 | 说明 |
|------|----------|----------|------|

## Formulas

**F1 — 信念值累积**

```
belief_value[dimension] += base_gain × choice_weight × relationship_modifier
```

| 变量 | 符号 | 类型 | 范围 | 说明 |
|------|------|------|------|------|
| base_gain | G | int | 1 ~ 5 | 行为基础信念值增益 |
| choice_weight | W | float | 0.5 ~ 2.0 | 选择重要度权重（对话=1.0，关键决策=2.0） |
| relationship_modifier | R | float | 0.8 ~ 1.5 | 羁绊加成（与对应路线角色羁绊高→增益更大） |

**输出范围**：单个维度累积值 0 ~ 100（归一化后）。每章结束时不重置。

**F2 — 分流判定**

```
dominant = max(果断, 隐忍, 探索)
second = second_max(果断, 隐忍, 探索)
branch_threshold = 15  # 最小差值

IF dominant - second >= branch_threshold:
    route = dimension_to_route(dominant)
ELSE:
    offer_choice(top_two_dimensions)
```

| 变量 | 符号 | 类型 | 范围 | 说明 |
|------|------|------|------|------|
| branch_threshold | T_branch | int | 10 ~ 25 | 主导维度与次高维度的最小差值 |
| dominant | V_max | int | 0 ~ 100 | 最高维度信念值 |
| second | V_2nd | int | 0 ~ 100 | 次高维度信念值 |

**输出**：确定的路线（if 差值足够）或玩家选择（if 差值不足）。

**F3 — 信念值对战斗的微妙影响**

```
belief_bonus = belief_value[dominant] / 100 × belief_power_factor
```

```
belief_power_factor = 0.05  # 最大 5% 影响
```

| 变量 | 符号 | 类型 | 范围 | 说明 |
|------|------|------|------|------|
| belief_value | V | int | 0 ~ 100 | 主导维度信念值 |
| belief_power_factor | F | float | 0.03 ~ 0.10 | 信念值对战斗的最大影响比例 |

**输出范围**：0% ~ 5% 的属性加成。信念值对战斗的影响是**微妙的氛围反馈**，不改变战斗平衡。

**F4 — 内容量估算**

```
total_playtime = shared_chapters × avg_chapter_time + route_chapters × avg_chapter_time + side_quest_count × avg_side_time
```

| 变量 | 符号 | 类型 | 范围 | 说明 |
|------|------|------|------|------|
| shared_chapters | N_shared | int | 5 ~ 7 | 所有路线共享的章节数 |
| route_chapters | N_route | int | 3 ~ 5 | 单条路线专属章节数 |
| avg_chapter_time | T_chapter | int | 45 ~ 90 min | 平均章节时长（含战斗） |
| side_quest_count | N_side | int | 8 ~ 15 | 单路线可完成支线数 |
| avg_side_time | T_side | int | 15 ~ 30 min | 平均支线时长 |

**输出范围**：单周目 ~15-25 小时。三周目全体验 ~50-70 小时。

## Edge Cases

- **If 玩家三个信念维度完全相同**（信念值差值 < threshold）：在分支点提供三选一的明确选择，而非自动判定。给出"你的内心仍在挣扎"的叙事提示。原因是平局说明玩家真正在做选择，应尊重玩家的主动权。

- **If 玩家在硬分支后试图改变信念方向**：硬分支（分支 2）后路线锁定不可逆转。后续行为仍积累信念值但不影响路线。信念值在二周目重置，不影响一周目的路线锁定。原因是叙事一致性比"后悔"更重要——选择=失去是核心原则。

- **If 羁绊角色因路线分流离开队伍**：离开角色触发专属"离别剧情"，其羁绊数据保留但标记为"已离队"。可通过多周目在同一路线中维持该羁绊。原因是角色离队是叙事的情感冲击点，不能被"召回"稀释。

- **If 隐藏分支条件未满足但玩家接近触发**：不显示任何提示。隐藏分支设计为二周目+特定信念组合的惊喜，提前提示会破坏发现感。

- **If 玩家跳过所有支线任务**：主线信念值来源足够完成分流判定。支线是"强化"而非"必需"。但跳过支线会导致某些角色无法招募、某些信念值维度偏低（可能影响分支选择范围）。

- **If 装备系统与路线专属装备冲突**（如玩家在路线 A 获得了路线 B 的装备）：路线专属装备通过掉落和奖励分配，不会出现跨路线获取的情况。如果通过多周目继承了装备，显示为"来自另一个时空的遗物"，不影响当前路线叙事。

- **If 存档中记录的选择与当前信念值不匹配**（存档损坏/修改）：以信念值数值为准，不依赖选择历史重建。信念值是路线判定的唯一依据，选择历史仅用于对话回放和成就判定。

## UI Requirements

- **对话系统**：支持角色立绘 + 对话文本 + 选择分支。选择项显示信念值倾向提示（颜色编码，非文字说明）
- **信念值面板**：展示三个维度的当前值，但不显示具体数字——用视觉填充度（如水墨浓淡）表达
- **章节标题卡**：每章开始时显示章节名和简短引言，2-3 秒后自动消失
- **分支确认界面**：显示当前信念值倾向，确认选择后不可撤销（硬分支）

## Acceptance Criteria

**GIVEN** 玩家完成主线第 1-4 章，**WHEN** 在第 5 章硬分支点检查信念值，**THEN** 系统根据信念值最高维度（或提供选择）正确分流到对应路线，分流结果与玩家整个旅程的行为选择一致。

**GIVEN** 玩家在对话中选择"立即进攻"选项，**WHEN** 该选择标记为果断维度，**THEN** 果断信念值增加 base_gain × choice_weight（默认 3×1.0 = 3），其他维度不变。

**GIVEN** 玩家处于朱红路线，**WHEN** 完成路线专属章节的战斗，**THEN** 掉落烈焰系列装备，不掉落其他路线的专属装备。

**GIVEN** 玩家与某路线角色羁绊达到 S 级，**WHEN** 到达对应路线的分支点，**THEN** 出现额外的羁绊专属选项（普通路线不提供）。

**GIVEN** 玩家三个信念维度差值 < 15，**WHEN** 到达分支点，**THEN** 提供前两个最高维度的手动选择，不自动判定。

**GIVEN** 玩家完成硬分支后继续积累信念值，**WHEN** 检查路线状态，**THEN** 路线锁定不变，信念值继续累积但不影响路线归属。

**GIVEN** 存档记录了完整的信念值和选择历史，**WHEN** 执行存档+读档循环，**THEN** 信念值、当前章节、路线归属、已完成的事件列表与存档前完全一致。

**GIVEN** 玩家跳过所有支线任务，**WHEN** 到达硬分支点，**THEN** 主线提供的信念值足以完成分流判定（不出现全部维度为 0 的情况）。

## Open Questions

- **OQ-1**：架空历史的具体朝代基础尚未确定（春秋战国？三国？唐宋？）。影响美术风格的角色服饰、建筑风格、武器设计。需要创意决策。
- **OQ-2**：预设主角的姓名、性别、背景设定尚未确定。影响叙事基调和玩家代入感。
- **OQ-3**：三路线的独立内容量是否平衡？如果某路线内容明显少于其他路线，需要调整章节分配。
- **OQ-4**：信念值是否对战斗有 5% 的实际属性影响（F3），还是纯叙事标识？影响游戏平衡设计。
- **OQ-5**：对话系统是否需要语音？影响音频预算和开发周期。MVP 阶段建议纯文本。
