SRPG 核心模块设计总纲

> 版本: v0.1 | 日期: 2026-04-16 | 状态: 初始决策完成

概述

本文档记录了SRPG各核心模块的设计方向决策。每个模块基于跨游戏研究报告的分析结论，经过逐项讨论确认。

设计愿景: 一款以架空中国历史为背景的深度养成SRPG，融合曹操传的属性潜质养成、皇骑的分支叙事、火纹的支援系统、FFT的职业自由度和兰斯的熟练度成长，以魂系试错哲学设计Boss战，以成就点数驱动多周目解锁。

核心设计原则: 系统互锁——每个子系统必须在策略和情感两个层面提供回报，且与其他子系统产生涌现式互动。

1. 世界观与题材

决策: 架空历史

以真实中国历史为骨架，加入虚构势力、人物和事件。玩家熟悉的历史框架降低理解门槛，架空元素提供叙事自由度。可融合多个朝代元素，不受史实束缚。

设计要点:

- 历史感是底色，不是枷锁
- 政治阴谋+道德灰色地带（皇骑式成熟叙事）
- 角色非脸谱化——每个角色有复杂动机

2. 叙事结构

决策: 信念值驱动三路线分支

融合皇骑L/N/C三路线+三角战略信念值系统。玩家行为累积影响立场（不只是剧情选择点），在关键节点根据累积值自然分流。

设计要点:

- 每条路线有独立故事+独立角色+不同道德代价
- 信念值让分支感觉"自然生长"而非"二选一"
- 选择=失去——每个选择都意味着失去其他可能性
- 3-5个关键分支点控制内容量

3. 属性系统

决策: 五维普通属性+隐藏属性+属性潜质+壁障突破+门槛奖励+属性碾压

3.1 普通属性


| 属性  | 名称  | 主要影响            |
| --- | --- | --------------- |
| 1   | 力量  | 物理攻击、负重         |
| 2   | 敏捷  | 速度、闪避、命中        |
| 3   | 体力  | 生命值、防御、耐久       |
| 4   | 智力  | 魔法攻击、技能效果       |
| 5   | 魅力  | 羁绊效率、NPC互动、特殊触发 |


3.2 隐藏属性


| 属性   | 影响         |
| ---- | ---------- |
| 幸运   | 暴击、掉落、随机事件 |
| 意志力  | 控制抵抗、濒死触发  |
| 异常抵抗 | 异常状态防御     |
| 灵魂强度 | 特殊机制（待细化）  |


3.3 核心机制

- 属性潜质: 每个属性有独立的"潜质"值，决定每级成长量。潜质与属性值完全分离
- 果子系统: 果子提升属性潜质（而非属性本身），间接影响长期成长
- 壁障突破: 属性达到特定阈值时触及壁障，需消耗稀缺资源或完成挑战突破，解锁新成长空间+属性奖励
- 门槛奖励: 属性达到特定门槛值时触发额外奖励（如力量达到50解锁重击能力）
- 真实属性/伪真实属性: 待细化——区分表面数值与实际生效数值
- 属性碾压: 特定技能在属性差距达到阈值时触发碾压效果

3.4 成长方式

- 第一周目: 确定成长——每级属性增长按潜质确定计算，无RNG
- 多周目: 可开启自由分配开关——确定增长值保留，玩家自选分配方向
- 每个职业体系对属性有增长要求

4. 职业系统

决策: 属性门槛+职业经验双重条件（混合式）

4.1 解锁条件

职业解锁需同时满足：

1. 属性门槛: 相关属性达到指定值（如"剑圣"需力量80+敏捷30）
2. 职业经验: 在相关基础职业中累积足够经验

4.2 职业分层


| 层级   | 可用性            | 示例                 |
| ---- | -------------- | ------------------ |
| 基础职业 | 一周目起始可用        | 战士、法师、弓手、盗贼、牧师、骑士等 |
| 进阶职业 | 一周目可解锁         | 基础职业的高级形态          |
| 特殊职业 | 多周目通过副本/特殊任务获得 | 独特职业，更复杂的技能树       |


4.3 设计要点

- 属性潜质投资方向直接影响职业发展路径
- 同一角色可通往多个职业但资源有限
- 职业切换有成本，不是随意切换
- 特殊职业作为多周目的成就点兑换项之一

5. 技能系统

决策: 熟练度驱动+位阶进阶+每10级进阶奖励

5.1 核心资源

- 熟练度: 每次战斗获得，独立于经验值。用于提升技能等级
- 每人的属性潜质与技能匹配度影响升级所需熟练度（因人而异）

5.2 技能分类


| 分类维度 | 类型        |
| ---- | --------- |
| 按来源  | 普通技能、职业技能 |
| 按使用  | 主动技能、被动技能 |


5.3 位阶系统

- 技能有位阶上限，达到当前位阶极限后无法通过熟练度继续提升
- 进阶下一位阶需满足特定条件：完成考核、隐藏属性达标等
- 位阶进阶是技能成长的关键节点

5.4 进阶奖励

- 关键节点（如9级升10级）所需熟练度大幅提升
- 每10级（10/20/30...）触发进阶奖励——从2-3个特性中选择一个
- 特性选择永久塑造技能风格（轮回乐园DNA）
- 示例：火球术Lv10可选"范围+1"或"伤害+20%"

5.5 技能树

- 普通职业：标准技能树
- 特殊职业/天赋角色：更复杂的技能树，分支更多

6. 装备系统

决策: 词缀+强化+分解+套装

6.1 品质等级

白 → 绿 → 蓝 → 紫 → 金

6.2 词缀系统（暗黑DNA）

- 每件装备有1-4条随机词缀（基于品质）
- 词缀种类由装备类型决定
- 词缀有数值范围，同类词缀数值随机
- 词缀影响战术行为（如"攻击附带冰冻"改变控场能力）

6.3 装备大厅


| 操作     | 说明                  |
| ------ | ------------------- |
| 强化     | 消耗材料+金币提升基础属性，有等级上限 |
| 附魔(重铸) | 消耗稀有材料重置一条随机词缀      |
| 打孔     | 添加宝石插槽，宝石提供固定属性加成   |
| 套装打造   | 收集材料打造套装部件，激活套装效果   |


6.4 强化风险机制

- 安全区: +1~+5安全，必定成功
- 风险区: +6以上有失败概率，失败可能损毁装备（永久消失或降级）
- 保护符: 防止损毁（但不防止失败）
- 祝福值保底: 连续失败积累进度，满值时下次必成

6.5 分解系统

装备可分解为养成资源（曹操传"装备→养成货币"的现代化）

6.6 套装效果

2件/4件/6件套提供战术性能力加成

7. 羁绊系统

决策: 深度羁绊（多类型+组合技+叙事+影响信念值）

7.1 好感度与等级

- 战斗中相邻/配合行动增加好感度（火纹式）
- 支援等级: C → B → A → S
- 每级解锁：战斗属性加成+组合技能+叙事对话
- 支援范围：战场上3格内生效

7.2 羁绊类型


| 类型  | 属性加成倾向 | 组合技风格 |
| --- | ------ | ----- |
| 战友  | 攻防均衡   | 协同攻击  |
| 师徒  | 成长加速   | 传授技能  |
| 宿敌  | 攻击强化   | 竞争激发  |
| 恋人  | 全面小幅提升 | 守护/牺牲 |


7.3 限制与策略

- 每人最多5条深度羁绊——不能和所有人都深度绑定
- 羁绊关系影响信念值走向
- "我和谁结羁绊"同时是情感和策略决策

8. 战术机制

决策: 克制三角+元素交互+高低差（三层叠加）

8.1 克制三角

兵种/武器类型间的克制关系（具体三角待细化）

8.2 元素交互

环境中的元素可相互作用：

- 火烧油 → 范围伤害
- 水导电 → 连锁伤害
- 风助火 → 火焰扩散
- 具体元素体系待细化

8.3 高低差

- 位置高度影响射程和命中（皇骑式）
- 高处→低处：射程+、命中+
- 低处→高处：射程-、命中-

8.4 设计目标

三层各自简洁，但组合后产生涌现式互动：

- 克制关系让编队有意义
- 元素交互让地形有意义
- 高低差让位置有意义

9. 回合制模式

决策: 速度序列制+自动战斗+加速

9.1 速度序列

- 所有单位根据速度属性排序行动
- 可视化行动顺序条，玩家可规划连招
- 速度投资有明确回报
- 与"敏捷"属性天然互锁

9.2 便捷功能


| 功能   | 说明              |
| ---- | --------------- |
| 自动战斗 | 开关式，AI自动操作我方单位  |
| 加速   | 开关式，加快动画和AI思考速度 |


10. Boss战设计

决策: 阶段检查点+部分保留 → 资源消耗兜底

10.1 主要机制（阶段检查点+部分保留）

- Boss分多阶段，每阶段切换时自动检查点
- 失败后从当前阶段重开
- 已削减Boss血量保留10-20%
- 首次失败后显示Boss模式提示
- 已学会的模式提示永久保留

10.2 兜底机制（资源消耗式）

- 多次失败后可选择"消耗资源再试"
- 消耗士气/补给等资源，但获得的经验/物品保留
- 玩家可选择"回去练级再来"或"消耗资源再试"

10.3 Boss设计七原则

1. 可学习的行动模式（魂系DNA）
2. 阶段切换改变战术需求（非纯数值提升）
3. 多解设计——没有唯一正确解（XCOM DNA）
4. 检查点机制（魂系+Roguelike DNA）
5. 部分失败恢复（Roguelike DNA）
6. 难度曲线双维度管理（数值+策略）
7. Boss作为叙事节点（皇骑DNA）

10.4 Boss类型


| 类型       | 目的     | 难度  | 阶段数 |
| -------- | ------ | --- | --- |
| 教学Boss   | 引入核心机制 | 低   | 1-2 |
| 叙事Boss   | 推动剧情   | 中   | 2   |
| 能力测试Boss | 检验养成   | 中高  | 2-3 |
| 巅峰Boss   | 综合考验   | 高   | 3+  |
| 隐藏Boss   | 极限挑战   | 极高  | 多阶段 |


11. 角色退场机制

决策: 故事退场式

- 角色战败不是"死亡"而是"受重伤退场"
- 触发独特的"退场剧情"——每个角色有专属退场故事
- 羁绊受影响（相关角色触发反应剧情）
- 可通过特定任务"招回"退场角色
- 提供关闭选项——关闭后变为"战败后恢复"模式

设计要点: 比纯死亡更有叙事价值，招回任务给玩家希望而非绝望

12. 多周目系统

决策: 成就点数兑换新周目开关

12.1 成就系统

- 每周目可达成多种成就（战斗/叙事/收集/隐藏等）
- 每个成就奖励成就点数
- 成就点数跨周目永久累积

12.2 新周目开关

开新周目时，用成就点数选购启用项：


| 开关类别 | 示例                        |
| ---- | ------------------------- |
| 属性成长 | 自由分配开关                    |
| 职业解锁 | 特殊职业可用                    |
| 难度倍率 | 2x/4x/8x/16x难度（敌人+奖励同步增强） |
| 内容解锁 | 特殊副本/隐藏角色招募条件             |
| 便利功能 | 待定                        |


12.3 设计要点

- 成就点数有限→玩家需要选择"这个周目我启用哪些"
- 高倍率难度消耗成就点但提供更强装备/资源
- 每个新周目都是"定制化的新体验"

13. 难度系统

决策: 固定难度+多周目倍率选择

- 第一周目固定难度曲线，不提供难度选择
- 多周目可选择难度倍率：2x / 4x / 8x / 16x
- 高倍率同步增强道具、装备等系统产出
- 难度倍率作为成就点消耗项

14. 视角与地图

决策: 2.5D斜45度方形网格

- 经典SRPG等距斜视角
- 方形网格（非六角格）
- 支持视角旋转
- 支持高低差地形
- 地图建议尺寸: 15x15 ~ 25x25

15. 战争迷雾

决策: 限定场景迷雾

- 大部分关卡信息全开
- 仅在特定关卡类型中启用：夜战、伏击、侦察任务
- 迷雾作为"特殊机制"丰富关卡多样性
- 与"侦察兵"类职业产生互锁

16. 美术风格

决策: HD-2D中国风

- 像素角色+3D场景+光影效果（三角战略式HD-2D）
- 融合中国山水画/水墨美学元素
- 视觉差异化：中国风HD-2D在市场上尚无先例

17. 基地系统

决策: 可升级基地+世界地图

17.1 基地功能区


| 功能区  | 对应系统             |
| ---- | ---------------- |
| 装备大厅 | 强化/附魔/打孔/套装打造/分解 |
| 训练场  | 技能熟练度训练、职业转换     |
| 市集   | 买卖装备/材料/果子       |
| 酒馆   | 羁绊事件、角色招募        |
| 情报室  | 关卡信息、世界地图、任务     |
| 其他   | 待定（根据需要扩展）       |


17.2 基地升级

- 基地整体可升级，解锁新功能区/功能
- 升级消耗资源（普通资源层）
- 与游戏进度/叙事阶段联动

17.3 世界地图

- 用于关卡选择和探索
- 显示可用战役/副本/事件
- 随叙事推进逐步开放

18. 资源经济

决策: 双层资源型

18.1 普通资源层（充裕）


| 资源   | 获取方式    | 用途          |
| ---- | ------- | ----------- |
| 金币   | 战斗奖励、出售 | 购买、强化基础消耗   |
| 基础材料 | 战斗掉落、探索 | 装备基础强化、基地升级 |
| 经验值  | 战斗      | 角色升级        |
| 熟练度  | 战斗      | 技能升级        |


18.2 稀有资源层（稀缺）


| 资源     | 获取方式             | 用途        |
| ------ | ---------------- | --------- |
| 果子     | Boss击败、特殊任务、装备分解 | 提升属性潜质    |
| 稀有材料   | 高难关卡、隐藏副本        | 装备附魔/高级强化 |
| 保护符    | 极稀有掉落            | 防止装备强化损毁  |
| 壁障突破资源 | Boss/特殊挑战        | 属性壁障突破    |
| 成就点数   | 成就达成             | 多周目开关兑换   |


18.3 设计原则

- 普通资源保证日常不焦虑
- 稀有资源创造核心策略取舍
- 两层资源消耗渠道分离，避免"一种资源争夺所有系统"

19. 多人/社交

决策: 纯单机，无社交功能

20. 目标平台

决策: PC优先

系统互锁图谱

核心系统之间的交叉涌现关系：

```
属性潜质 ──→ 职业解锁(门槛)
    │              │
    │         职业经验 ──→ 职业进阶
    │              │
果子分配         技能熟练度 ──→ 技能位阶 ──→ 每10级特性
    │              │
    ↓              ↓
壁障突破      装备词缀 ──→ 战术行为改变
    │              │
    │         装备分解 ──→ 养成资源(果子等)
    │              │
属性碾压      套装效果 ──→ 编队策略
    │              │
    ↓              ↓
信念值 ←── 羁绊系统 ──→ 组合技 ──→ 战斗效果
    │              │
    ↓              ↓
三路线分支    角色退场/招回 ──→ 叙事内容
    │
    ↓
成就点数 ──→ 多周目开关 ──→ 特殊职业/难度倍率/副本
```
