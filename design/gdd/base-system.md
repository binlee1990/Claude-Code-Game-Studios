# 基地系统

> **Status**: Designed (pending review)
> **Author**: binlee1990 + agents
> **Last Updated**: 2026-04-22
> **Implements Pillar**: 系统互锁（基地连接资源经济、装备、羁绊、叙事，是系统间交互的枢纽）

## Overview

基地系统管理玩家的可升级据点和世界地图，作为战斗之外的策略层。基地包含多个功能区（装备大厅、训练场、市集、酒馆、情报室），每个功能区对应一个或多个游戏子系统。基地升级消耗普通资源，解锁新功能区或强化现有功能。世界地图用于关卡选择和探索，随叙事推进逐步开放。

## Player Fantasy

- **归巢的安宁感**：战斗后回到基地，感受"这里有安全的仓库、熟悉的NPC、可以规划下一步"的掌控感
- **建设的成就感**：从简陋营寨到繁荣据点，每次升级都是玩家投入的可见回报
- **策略规划台**：在基地分配资源、调整装备、规划羁绊，是战斗前的"战争会议室"

## Detailed Design

### Core Rules

**规则 1 — 功能区**

| 功能区 | 对应系统 | 解锁条件 | 功能 |
|--------|---------|---------|------|
| 装备大厅 | 装备系统 | 初始 | 强化/附魔/打孔/套装打造/分解 |
| 训练场 | 技能系统、职业系统 | 初始 | 技能熟练度训练、职业转换 |
| 市集 | 资源经济 | 初始 | 买卖装备/材料/果子 |
| 酒馆 | 羁绊系统、叙事系统 | 基地 Lv2 | 羁绊事件、角色招募、NPC对话 |
| 情报室 | 叙事系统 | 基地 Lv3 | 关卡信息、世界地图、任务列表 |

**规则 2 — 基地升级**

| 等级 | 升级消耗 | 解锁内容 |
|------|---------|---------|
| Lv1（初始） | — | 装备大厅、训练场、市集（基础功能） |
| Lv2 | 500 金币 + 20 基础材料 | 酒馆解锁；市集扩展（稀有物品） |
| Lv3 | 1500 金币 + 50 基础材料 | 情报室解锁；训练场扩展 |
| Lv4 | 3000 金币 + 100 基础材料 | 所有功能区强化；新外观 |
| Lv5 | 5000 金币 + 200 基础材料 + 5 稀有材料 | 全功能解锁；基地专属 Boss 挑战 |

**规则 3 — 世界地图**

- 用于关卡选择和探索
- 显示可用战役/副本/事件
- 随叙事推进逐步开放（新章节解锁新区域）
- 已完成的关卡可重玩（不同难度/不同队伍组合）

**规则 4 — 行动点系统**

在基地中，每章有有限的行动点用于功能区互动：

| 行动 | 消耗 | 说明 |
|------|------|------|
| 装备强化 | 0 | 无限制操作 |
| 技能训练 | 1 点 | 每章有限训练次数 |
| 羁绊对话 | 1 点 | 每章有限对话次数 |
| 购买/出售 | 0 | 无限制交易 |
| 关卡选择 | 1 点 | 进入战斗 |

每章开始时恢复 5 个行动点。

### Interactions with Other Systems

| 系统 | 接口 | 方向 | 说明 |
|------|------|------|------|
| 装备系统 | 装备大厅操作 | 双向 | 强化/附魔/分解 |
| 技能系统 | 训练场操作 | 双向 | 熟练度训练 |
| 职业系统 | 训练场操作 | 双向 | 职业转换 |
| 资源经济 | 市集操作 | 双向 | 买卖交易 |
| 羁绊系统 | 酒馆操作 | 双向 | 羁绊事件触发 |
| 叙事系统 | 情报室/酒馆 | 双向 | 任务/剧情推进 |
| 存档系统 | 基地数据持久化 | 基地→存档 | 基地等级/解锁状态 |

## Formulas

**F1 — 基地升级消耗**

```
upgrade_cost = base_cost × (current_level ^ 1.5)
```

| 等级 | 金币 | 材料 |
|------|------|------|
| 1→2 | 500 | 20 |
| 2→3 | 1500 | 50 |
| 3→4 | 3000 | 100 |
| 4→5 | 5000 | 200 + 5 稀有 |

**F2 — 行动点恢复**

```
action_points = min(max_points, current_points + points_per_chapter)
```

max_points = 5, points_per_chapter = 5（每章全额恢复）

## Edge Cases

- **If 玩家行动点耗尽但未进入主线战斗**：仍可使用无消耗功能区（装备大厅/市集），但无法训练或触发羁绊事件。
- **If 基地升级所需材料不足**：升级按钮灰显，显示缺少的材料和获取建议。
- **If 路线分流导致基地 NPC 变化**：部分 NPC 跟随路线变化（某些 NPC 只在特定路线出现在基地）。

## Dependencies

### 上游
- **资源经济**：升级消耗金币和材料
- **装备系统**：装备大厅操作

### 下游
- **羁绊系统**：酒馆触发羁绊事件
- **叙事系统**：情报室推进剧情
- **存档系统**：保存基地状态

## Tuning Knobs

| Knob ID | 名称 | 默认值 | 安全范围 | 极端行为 |
|---------|------|--------|----------|----------|
| `TK-BASE-01` | 每章行动点 | 5 | 3~8 | 过少=限制过多；过多=无策略性 |
| `TK-BASE-02` | Lv2 升级金币 | 500 | 200~1000 | 过低=太快升级；过高=拖延 |
| `TK-BASE-03` | 训练消耗行动点 | 1 | 0~2 | 0=无限制训练；2=训练太贵 |

## Visual/Audio Requirements

- 基地使用 2.5D 等距视角（与战斗场景一致的 HD-2D 风格）
- 每个功能区有独特视觉标识（图标/旗帜/建筑风格）
- 升级时有建设动画和音效
- 基地整体随等级提升外观变化（从简陋到宏伟）

## UI Requirements

- 基地主界面：功能区列表 + 当前等级 + 资源余额 + 行动点
- 世界地图：章节进度 + 可用关卡 + 已完成标记
- 功能区内部：对应系统的操作界面

## Acceptance Criteria

**GIVEN** 基地 Lv1 且资源充足，**WHEN** 升级到 Lv2，**THEN** 消耗 500 金币 + 20 材料，解锁酒馆，市集扩展。

**GIVEN** 每章开始，**WHEN** 检查行动点，**THEN** 恢复至 5 点。

**GIVEN** 行动点已耗尽，**WHEN** 尝试训练技能，**THEN** 操作被拒绝，提示"行动点不足"。

**GIVEN** 世界地图上已完成的关卡，**WHEN** 选择重玩，**THEN** 可选择不同难度和队伍组合进入战斗。
