# 视角与地图

> **Status**: In Design
> **Author**: user + agents
> **Last Updated**: 2026-04-16
> **Implements Pillar**: 系统互锁——视角与地图是游戏世界的视觉呈现

## Overview

视角与地图是游戏世界的视觉呈现层——它决定了玩家"在哪里看"和"看到什么"。

游戏采用**2.5D斜45度等距视角**：
- 经典SRPG等距视角，斜45度角俯视战场
- 方形网格（15×15～25×25格）
- 支持视角旋转（4个固定角度）
- 支持高低差地形（低地/平地/高地）
- 像素角色+3D场景+光影效果（三角战略式HD-2D）

这种视角融合了经典SRPG的可读性和现代HD-2D的视觉美感。

## Player Fantasy

玩家在视角与地图中的核心感受是**"我身处一个鲜活的世界"**——战场不是抽象的格子，而是一个有氛围、有深度、有美感的环境。

**HD-2D视觉**带来的是**审美的满足感**。当玩家看着中国山水画风格的战场背景，看着像素角色在3D场景中移动时——这种**高品质的视觉呈现**让游戏从"玩法机器"变成了"艺术作品"。

**等距视角**带来的是**信息的清晰感**。当玩家能一眼看清战场全局，当每个单位的位置、高度、状态都清晰可见时——这种**信息的有效传递**是策略游戏的基础。

**具体锚点：**
- 战场上的高山流水、风云变幻 → 世界沉浸感
- 视角旋转后看到的全新构图 → 探索的美感
- 高低差地形带来的视觉层次 → 深度的美感

## Detailed Design

### Core Rules

**C.1 视角类型**

| 视角类型 | 描述 |
|---------|------|
| 2.5D斜45度等距视角 | 固定45度角俯视，可旋转4个方向 |

**C.2 视角旋转**

支持4个固定旋转角度：
- 0度（默认）
- 90度
- 180度
- 270度

旋转操作：快捷键或UI按钮触发

**C.3 地图规格**

| 规格 | 尺寸 | 用途 |
|------|------|------|
| 标准 | 15×15格 | 普通关卡 |
| 大型 | 20×20格 | 精英关卡 |
| 超大 | 25×25格 | Boss战或特殊关卡 |

**C.4 网格系统**

- 方形网格（非六角格）
- 每格1个单位标准占据面积
- 网格用于移动判定和位置计算
