# 羁绊系统

> **Status**: Designed (pending review)
> **Author**: binlee1990 + agents
> **Last Updated**: 2026-04-22
> **Implements Pillar**: 系统互锁（羁绊在策略和情感两个层面提供回报）

## Overview

羁绊系统管理角色之间的关系深度，通过战斗配合和叙事互动积累好感度，解锁战斗属性加成、组合技能和专属叙事内容。每个角色最多 5 条深度羁绊，使"和谁结羁绊"同时是策略决策和情感选择。羁绊类型（战友/师徒/宿敌/恋人）决定属性加成倾向和组合技风格。

系统引用总纲决策：深度羁绊（多类型+组合技），参考火焰纹章支援系统。

## Player Fantasy

- **战友之约**：玩家感受到角色之间真实的情感联系——不是数值加成，而是两个角色在战场上并肩作战数章后自然产生的信任
- **策略与情感交织**：选择和谁结羁绊同时影响战斗力（组合技强度）和叙事走向（信念值偏向）
- **限制即深度**：每人最多 5 条羁绊，意味着"不能和所有人都深度绑定"——选择即放弃

## Detailed Design

### Core Rules

**规则 1 — 好感度与支援等级**

好感度通过战斗配合和叙事互动累积。支援等级分为 C → B → A → S 四级。

| 等级 | 好感度阈值 | 解锁内容 |
|------|-----------|----------|
| C | 50 | 基础属性加成 + 支援对话 1 |
| B | 150 | 增强属性加成 + 支援对话 2 |
| A | 350 | 组合技能 + 支援对话 3 |
| S | 600 | 最大加成 + 专属剧情 + 组合技强化 |

**规则 2 — 好感度来源**

| 来源 | 好感度增益 | 条件 |
|------|-----------|------|
| 战斗中相邻行动 | +2/回合 | 两角色距离 ≤3 格 |
| 协助击杀 | +5 | A 击杀 B 协攻的目标 |
| 保护行动 | +8 | A 替 B 承受伤害 |
| 支援对话 | +15~25 | 在基地触发（消耗 1 个基地行动点） |
| 羁绊专属事件 | +30~50 | 剧情中自动触发 |

**规则 3 — 羁绊类型**

| 类型 | 属性加成倾向 | 组合技风格 | 触发倾向 |
|------|------------|-----------|---------|
| 战友 | 攻防均衡 +3% | 协同攻击（双人合击） | 共同参战次数多 |
| 师徒 | 成长加速（经验+10%） | 传授技能（临时借用对方技能） | 等级差 >5 且共同行动 |
| 宿敌 | 攻击强化 +5% | 竞争激发（一方受击另一方加攻） | 战斗中竞争击杀数 |
| 恋人 | 全属性 +2% | 守护/牺牲（替对方抵挡致命伤害） | 特定剧情选择 + 高好感度 |

类型由系统根据互动模式自动判定，不手动选择。判定权重：互动行为占比 >60% 则倾向对应类型。

**规则 4 — 组合技能（A 级解锁）**

组合技消耗双方行动点，效果由羁绊类型决定：

| 类型 | 组合技名 | 效果 | 冷却 |
|------|---------|------|------|
| 战友 | 协力一击 | 双人合击，伤害 ×1.5，无视目标 20% 防御 | 3 回合 |
| 师徒 | 技能传授 | 角色临时获得搭档的一项技能（持续 3 回合） | 5 回合 |
| 宿敌 | 竞争觉醒 | 一方受击后另一方获得 +30% 攻击，持续 2 回合 | 4 回合 |
| 恋人 | 誓约守护 | 搭档受到致命伤害时，自动替挡一次（消耗自身 30% HP） | 每场战斗 1 次 |

**规则 5 — 羁绊上限与策略**

每人最多 5 条深度羁绊（达到 B 级以上）。达到上限后，新羁绊增长速度减半。玩家可以选择"断绝"一条羁绊（降至 C 级），但对方好感度清零且可能触发负面剧情。

**规则 6 — 羁绊与信念值**

与特定路线角色的深度羁绊影响信念值走向：
- 与朱红路线角色达到 A 级 → 果断维度 +5
- 与墨黑路线角色达到 A 级 → 隐忍维度 +5
- 与青绿路线角色达到 A 级 → 探索维度 +5

### States and Transitions

```
无羁绊 → C级(好感度≥50) → B级(≥150) → A级(≥350) → S级(≥600)
  ↑           ↓
  └── 断绝(降至C, 好感清零)
```

### Interactions with Other Systems

| 系统 | 接口 | 方向 | 说明 |
|------|------|------|------|
| 属性系统 | 羁绊属性加成 | 羁绊→属性 | 支援范围内属性加成（被动自动生效，无需玩家手动触发） |
| 战斗结算 | 协助击杀/保护事件 | 战斗→羁绊 | 战斗行为产生好感度 |
| 叙事系统 | 羁绊专属事件、信念值影响 | 双向 | 羁绊推进叙事；叙事触发羁绊事件 |
| 角色管理 | 羁绊列表查询 | 角色→羁绊 | 角色详情显示羁绊状态 |
| 存档系统 | 羁绊数据持久化 | 羁绊→存档 | 保存所有羁绊等级和类型 |

## Formulas

**F1 — 好感度累积**

```
affinity_gain = base_gain × proximity_modifier × type_affinity
```

| 变量 | 符号 | 类型 | 范围 | 说明 |
|------|------|------|------|------|
| base_gain | G | int | 2 ~ 50 | 行为基础好感增益 |
| proximity_modifier | M_prox | float | 0.5 ~ 1.0 | 距离修正（≤3格=1.0, >3格=0.5，曼哈顿距离，见 tactical-mechanism.md） |
| type_affinity | M_type | float | 0.8 ~ 1.3 | 类型匹配修正 |

**F2 — 属性加成计算**

```
bond_bonus = base_stat × (bond_level_bonus + type_bonus)
```

| 等级 | bond_level_bonus | S级额外 |
|------|-----------------|---------|
| C | +1% | — |
| B | +2% | — |
| A | +3% | — |
| S | +5% | +组合技强化 |

**F3 — 羁绊上限影响**

```
if active_bonds >= MAX_BONDS:
    affinity_rate *= 0.5
```

MAX_BONDS = 5（每人最多 5 条深度羁绊）

## Edge Cases

- **If 两个角色同时触发保护行动**：先触发保护的角色获得好感度，后触发的无效。按行动顺序决定。
- **If 羁绊角色因路线分流离队**：羁绊数据保留但标记为"已离队"，不提供战斗加成。多周目中可恢复。
- **If 好感度达到 S 级后继续累积**：好感度封顶于 999，不再增长。多余好感度不转换。
- **If 断绝羁绊后重新培养**：好感度从 0 重新开始，类型可能改变（取决于新的互动模式）。

## Dependencies

### 上游
- **属性系统**：属性值用于计算羁绊加成
- **战斗结算**：战斗行为是好感度主要来源

### 下游
- **叙事系统**：羁绊事件推进叙事，深度羁绊影响信念值
- **角色管理**：角色详情显示羁绊状态
- **存档系统**：保存全部羁绊数据
- **HP系统**：恋人羁绊"誓约守护"消耗 30% HP 以战斗内 max_hp 为基准（`hp_cost = max_hp × 0.30`）

## Tuning Knobs

| Knob ID | 名称 | 默认值 | 安全范围 | 极端行为 |
|---------|------|--------|----------|----------|
| `TK-BOND-01` | 支援生效范围 | 3格 | 1~5 | 过小=难以触发；过大=失去策略性 |
| `TK-BOND-02` | 深度羁绊上限 | 5 | 3~8 | 过少=选择压力过大；过多=无策略性 |
| `TK-BOND-03` | C级好感阈值 | 50 | 20~100 | 过低=羁绊廉价；过高=难以建立 |
| `TK-BOND-04` | S级好感阈值 | 600 | 300~1000 | 过低=轻易满级；过高=不可能达到 |
| `TK-BOND-05` | 组合技冷却 | 3~5回合 | 1~10 | 过短=滥用；过长=无用 |
| `TK-BOND-06` | 羁绊信念值加成 | +5 | 0~10 | 过高=羁绊绑架叙事；过低=无叙事影响 |

## Visual/Audio Requirements

- 羁绊触发时显示连接光效（两角色间的光线，颜色匹配羁绊类型）
- 组合技触发时有专属动画（双人合作动作）
- S 级羁绊达成时有特殊的仪式性视觉效果

## UI Requirements

- 战斗中显示支援范围内的羁绊连线
- 角色详情页显示羁绊列表（类型、等级、好感度进度条）
- 基地中可查看和管理羁绊关系网络图

## Acceptance Criteria

**GIVEN** 两角色在战场上距离 ≤3 格，**WHEN** 协助击杀同一目标，**THEN** 好感度增加 5 × proximity_modifier。

**GIVEN** 两角色好感度达到 150，**WHEN** 系统检查支援等级，**THEN** 升级至 B 级，解锁增强属性加成和支援对话 2。

**GIVEN** 羁绊达到 A 级，**WHEN** 触发组合技，**THEN** 根据羁绊类型执行对应效果（战友=协力一击，师徒=技能传授等），消耗双方行动点。

**GIVEN** 角色已有 5 条深度羁绊，**WHEN** 与新角色互动积累好感度，**THEN** 好感度增长速度减半。

**GIVEN** 完整存档+读档循环，**WHEN** 检查羁绊数据，**THEN** 所有羁绊等级、类型、好感度与存档前完全一致。

## Open Questions
