> 版本：v1.0
> 最后更新：2026-05-05
> 关联文档：`design/gdd/game-concept.md`（§3 玩家幻想、§4 设计支柱、§11 MVP 定义）、`design/gdd/cultivation-system.md`、`design/gdd/zone-system.md`

## 1. 世界观前提

天地有灵气，万物可修行。玩家出身于苍玄山脚下一个被遗忘的凡人村落——**青石村**。村中老人传言，村后山中有上古修士遗留的洞府入口，只是百年来无人得见。一日，你在村外采药时误触一道残破禁制，一缕天地灵气灌入体内……

从此，你不再是凡人。

> **叙事定位**：本作不追求宏大开场和复杂剧情线，而是用极简开局（参见 game-concept.md §4.6 "渐进叙事展开"）让玩家自己"发现世界"。这个世界是修仙门派的边缘地带，远离大能博弈，恰好适合一个散修的静默成长。

## 2. 境界命名体系

### 2.1 核心境界命名（MVP + 近端预留，共 5 境）

遵循修仙小说传统的境界递进，从凡人到元婴构成第一幕的命名弧线。当前 Lv.1–30 MVP 实装只落到**筑基期**（`level-system.md` / `LevelSystem._load_default_realms()`：凡人 Lv.1、炼气 Lv.10、筑基 Lv.30）；**金丹期、元婴期在本档中只作为 Phase 2+ 叙事牵引和 UI 图标命名预留**，不写入 Sprint 12 的数值验收。

| 序号 | 境界名 | 境界简述 | 叙事意义 |
|------|--------|----------|----------|
| 0 | **凡人** | 尚未引气入体，筋骨凡胎。没有修为，没有神通，只有一具血肉之躯和对"山那边"的好奇。 | 开局状态。让玩家从零感受"变强"的起点。 |
| 1 | **炼气期** | 引气入体，灵气在经脉中流转。五感渐锐，力能碎石，百年寿元初启。 | 第一个"我变强了"的体感。解锁修炼系统全部姿态。 |
| 2 | **筑基期** | 道基铸就，肉身脱胎换骨。灵气液化，可在丹田聚气成旋。神识初开，可感知周围灵气浓淡。 | Lv.30 MVP 终局仪式。玩家完成第一阶段闭环，并看到金丹/装备刷宝等 Phase 2+ 目标。 |
| 3 | **金丹期** | 丹田结丹，灵气质变。神通初显——飞剑、御物、简单法术皆可施展。寿元增至五百载。 | Phase 2+ 近端目标。作为荒殒战场之后的下一段期待，不作为 Lv.1–30 MVP 结算目标。 |
| 4 | **元婴期** | 金丹碎裂，元婴初成。神魂可短暂离体，灵气掌控精细入微。战力质变，真正踏入"修仙者"行列。 | Phase 2+ 中期目标。为突破至化神期积蓄力量。 |

### 2.2 Phase 2+ 扩展境界（预留命名空间）

以下境界已确认命名，但不在 MVP 范围内。对应 game-concept.md §12 阶段路线图中的后续阶段。

| 序号 | 境界名 | 阶段 | 叙事定位 |
|------|--------|------|----------|
| 5 | **化神期** | Phase 2+ | 元神与天地交感，可分化神念。解锁宗门系统。 |
| 6 | **合体期** | Phase 3+ | 元神与肉身合一，战力暴增。解锁宠物/器灵联动。 |
| 7 | **大乘期** | Phase 4+ | 凡界无敌，开始接触天地法则。解锁天劫系统。 |
| 8 | **渡劫期** | Phase 4+ | 天劫降临，渡之则升仙。解锁飞升系统。 |
| 9 | **飞升/真仙** | Phase 6+ | 踏入仙界，开启新世界层。解锁世界层级系统（190）。 |
| 10+ | **天仙 · 金仙 · 太乙 · 大罗 · 混元 · 合道** | Phase 7-9 | 诸天万界的顶点，对应 game-concept.md §5 中的法则重置、世界规则改写等终局系统。 |

### 2.3 小境界细分（MVP 为各境界保留）

每个大境界内可设前、中、后、圆满四小层，为未来扩充数值深度预留空间。MVP 期间为避免信息过载，仅展示大境界名称，小境界作为内部数值分段存在（如"炼气前期 / 炼气中期 / 炼气后期 / 炼气圆满"）。

## 3. MVP 三区域命名与主题

对应 game-concept.md §11 的"3 个挂机区域"与 zone-system.md 的区域定义。三个区域构成一个完整的"边缘修士 → 散修探索者 → 遗迹寻宝者"的叙事弧。

### 3.1 区域 1：青丘山林（新手/入门）

| 维度 | 描述 |
|------|------|
| **名称** | 青丘山林 |
| **解锁条件** | 开局默认解锁（无门槛） |
| **推荐境界** | 凡人 → 炼气前期 |
| **环境描述** | 青石村外连绵的丘陵地带，古木参天，野草丛生。山间溪流潺潺，偶见废弃药田和猎人木屋的残迹。空气中灵气稀薄但纯净，适合初学者感受天地之息。白天鸟鸣兽走，夜晚萤火点点，一片祥和中透着被遗忘的荒凉。 |
| **敌人类型** | 山精（石灵、木魅）——被微弱灵气浸染的自然生灵；野妖（狐妖幼体、蛇蜥）——初开灵智的小妖兽；游魂（残念、迷途灵体）——死在附近的凡人执念所化。所有敌人攻击模式简单，伤害低，用于玩家适应自动战斗。 |
| **氛围关键词** | 宁静、荒凉、探索感、安全的野外 |
| **掉落倾向** | Sprint 12 入账基础药材（止血草、聚灵花）与低级灵石；凡品装备胚仅作 Phase 2+ 叙事线索 |
| **叙事钩子** | 山中偶有奇异光柱一闪而逝——那是上古禁制偶尔泄出的灵气余波。你只是在采药，却无意间触碰了命运。 |

### 3.2 区域 2：幽墟灵谷（进阶）

| 维度 | 描述 |
|------|------|
| **名称** | 幽墟灵谷 |
| **解锁条件** | Lv.5 + atk ≥ 180（数值门槛；参见 mvp-content-progression.md §2.1） |
| **推荐境界** | 炼气期前后 |
| **环境描述** | 青丘山深处一条被遗忘的裂谷。谷中灵气浓度骤升，却混杂着一丝诡异的寒意。千年古藤缠绕着崩裂的石阶，石壁上铭刻着残缺的阵纹。谷底有一汪灵泉，泉水倒映的不是天空，而是不知何处的星图。有散修在此结庐苦修，却鲜有人能活着走出谷之尽头。 |
| **敌人类型** | 灵兽（灵雾狼、赤冠鹤）——长期吸收谷中灵气而变异的兽类；散修敌对者（走火入魔的苦修者、贪图玩家资源的投机者）；阵灵（残阵显化的能量体）——被废弃阵法激活的自动防御。MVP 只用 HP/ATK/DEF/SPD 等基础属性制造差异；技能、属性克制和战法配置留作 Phase 2+。 |
| **氛围关键词** | 神秘、隐忧、灵气的双刃剑、人迹罕至 |
| **掉落倾向** | Sprint 12 入账中等药材（凝血草、灵泉精华）与灵石；精良装备胚、基础功法残页仅作 Phase 2+ 叙事线索 |
| **叙事钩子** | 星图倒映的灵泉底，似乎藏着什么东西——或许是上古修士留下的传承，又或许，是他特意封印的隐患。 |

### 3.3 区域 3：荒殒战场（挑战）

| 维度 | 描述 |
|------|------|
| **名称** | 荒殒战场 |
| **解锁条件** | Lv.15 + atk ≥ 1,600（数值门槛；参见 mvp-content-progression.md §2.1） |
| **推荐境界** | 炼气后期 → 筑基期 |
| **环境描述** | 幽墟灵谷尽头，裂谷豁然开朗——眼前是一片焦黑的古战场。千年前，两位大能在此死斗，余波将方圆百里化为死地。破碎的法宝碎片散落荒原，巨大的剑痕切开了山脊，残存的法则之力让此地的灵气狂暴而混乱。焦土之下，偶尔能听见金铁交鸣的幻音，那是历史不肯消散的回响。 |
| **敌人类型** | 法器傀儡（剑傀、阵卫）——古战场遗留的战斗造物，仍在执行主人的最后命令；怨煞（战死修士的执念聚合体）——集合了不甘与仇恨的灵体；残魂（陨落大能的神念碎片）——稀有敌人预留。MVP 用更高基础属性与守护灵精英标记表达挑战；范围攻击、精英词缀和专属掉落留作 Phase 2+。 |
| **氛围关键词** | 肃杀、危险、史诗遗迹、逆天机缘 |
| **掉落倾向** | Sprint 12 入账高阶药材（龙血苔、残阳花）与中高级灵石；装备胚、完整功法残卷、古修遗物材料仅作 Phase 2+ 叙事线索 |
| **叙事钩子** | 战场上最深的那道剑痕之下，掩埋着一柄折断的古剑。它已经死了——但当你靠近时，还能感受到剑中器灵残存的一息执念，它在等一个能重新举起它的人。 |

### 3.4 区域过渡感设计

| 过渡 | 玩家感受到的变化 |
|------|------------------|
| 区域 1→2 | 从"我在野外捡垃圾"到"我进入了一个真正危险但有机遇的地方"。敌人不再是一刀秒，灵气浓度变化带来视觉差异（HUD 可展示灵气浓度条）。解锁"区域选择"界面。 |
| 区域 2→3 | 从"散修的日常"到"站在历史的骸骨上"。环境色调从青绿转为暗红/灰黑，背景音乐从空灵转低沉。敌人开始有名字（不只是"灵雾狼"而是"灵雾狼·精英"），掉落出现第一件紫色/金色品质。 |

## 4. 开局文本

### 4.1 标题画面氛围文案

> 天地不仁，万物刍狗。
> 唯修者，逆天而行。

*（设计说明：两句化用《道德经》"天地不仁，以万物为刍狗"，翻转后半句为修仙世界观的核心信念——"逆天而行"。字体使用水墨书法风格，背景为青丘山林的静态远景。）*

### 4.2 首次进入修炼界面时的引导文本

> 你盘膝坐定，闭目凝神。
> 山风穿过窗棂，带来第一缕若有若无的天地灵气。

*（设计说明：不解释"修炼系统怎么用"，只用沉浸式文字暗示"你已经开始修炼了"。玩家看到灵气数值开始跳动，自然理解系统行为。参见 game-concept.md §4.6 "渐进 UI 与修仙世界展开"。）*

### 4.3 首次触发战斗时的叙事钩子

> 村中猎户警告过你：山中有妖。
> 此刻，灌木丛后的那双猩红眼睛，终于让你明白了这句话的分量。

*（设计说明：战斗不是"无来由杀怪"，而是警告兑现。叙事不定义敌人强弱，让战斗系统本身的胜负来决定。首次战斗应为弱敌，保证玩家胜利，建立"我可以应对"的信心。）*

## 5. 核心术语表

### 5.1 MVP 术语

| 术语 | 定义 | 系统归属 |
|------|------|----------|
| **灵气** (`lingqi`) | 天地间流动的基础能量。修炼者通过吸收灵气转化为自身修为。也是使用法术、驱动法宝的基础消耗资源。MVP 中灵气为自动产出 + 手动点击获得。 | 资源系统 (46)、修炼系统 (48) |
| **修为** (`xiuwei`) | 修炼者体内积累的"道行"，由灵气凝练而成。修为累积到一定程度即可尝试境界突破。MVP 中修为是境界突破的主要门槛资源。 | 资源系统 (46)、修炼系统 (48) |
| **灵石** (`lingshi`) | 蕴含高浓度灵气的天然矿石，修仙界的通用货币。MVP 中灵石用于购买基础升级和作为少量消耗资源。 | 资源系统 (46) |
| **境界** (`realm`) | 修仙者的实力层级，从凡人到元婴。每个境界对应不同的战力上限、系统解锁权限和叙事地位。参见 §2。 | 境界绑定系统 (72) |
| **洞府** (`cavern`) | 修炼者居住和修行的私人场所，通常建于灵气充沛之地。MVP 中洞府是解锁后的常驻界面，展示当前修炼状态和基础设施。Phase 2+ 扩展为完整建筑系统 (141-142)。 | 洞府系统 (141, Phase 2+) |
| **修炼** (`cultivation`) | 吸收灵气、凝练修为的过程。MVP 支持四种姿态：打坐（均衡）、冥想（偏灵气产出）、凝练（灵气转修为）、闭关（高倍率，预留）。 | 修炼系统 (48) |
| **挂机** (`auto-battle / idling`) | 队伍在选定区域自动循环战斗，玩家无需操作。在线时显示战斗日志，离线时通过时间差值结算。 | 挂机探索系统 (99)、离线收益结算系统 (61) |
| **离线** (`offline`) | 玩家退出游戏后，系统通过时间戳差值为玩家补发挂机收益。MVP 离线收益包含灵气、经验、灵石和基础药材；装备收益为 Phase 2+。 | 离线模拟内核 (4)、离线收益结算系统 (61) |
| **区域** (`zone`) | 可挂机的敌人生成区域，不同区域有不同敌人池、掉落表和推荐境界。MVP 共 3 区域。参见 §3。 | 区域系统 (97) |
| **掉落** (`loot / drop`) | 击败敌人后获得的物品。Sprint 12 只承诺药材、灵石、经验类收益；装备、气运和复杂加成进入 Phase 2+。 | 掉落系统 (113) |
| **战斗** (`combat`) | MVP 为半自动战斗：普通攻击自动释放，玩家主要选择挂机区域并观察结果。技能与预设战法条件为 Phase 2+ 扩展。 | 半自动战斗系统 (89) |

## 6. 渐进叙事展开设计

对应 game-concept.md §4.6 "渐进叙事展开"支柱，以及系统 33 "渐进 UI 解锁"。

### 6.1 开局：只有修炼

玩家首次进入游戏时，看到的世界极其简单：

```text
[标题画面] → "开始修炼" → [修炼界面]
```

- **仅开放系统**：修炼（基础打坐）、资源显示（灵气、修为）
- **仅显示信息**：当前境界（凡人）、每秒灵气产出、手动修炼按钮
- **隐藏的所有系统**：战斗、区域选择、离线结算界面、装备、背包、存档（存档在后台自动运作但不展示界面）

叙事体验：玩家看到的不是一个"菜单"，而是一个有沉浸感的修炼画面。灵气数字缓慢跳动，修为条缓慢填充。几分钟后，灵气达到某个阈值（如 100），界面底部浮现一行淡入文字：

> "体内灵气似乎冲破了什么关窍……你隐约感知到山中有异动。"

此时解锁战斗系统和区域 1。

### 6.2 三区域叙事节奏

| 阶段 | 区域 | 叙事节奏 | 解锁触发 | 玩家体感 |
|------|------|----------|----------|----------|
| **发现** | 青丘山林 | 从修炼到战斗的首次扩展：世界变大了。玩家不是被推去战斗，而是"灵气够了自然地感知到可以去的地方"。 | 灵气 ≥ 100 或首次手动修炼 10 次 | "原来我还能走出这片林子？" |
| **熟悉** | 幽墟灵谷 | 区域 1 已被探索，敌人已成"刷材料的地方"。区域 2 的解锁是战力成长的自然结果，叙事上用"灵气浓度吸引"驱动。 | Lv.5 + atk ≥ 180 | "这里比我想象的更深、更危险——但也更富饶。" |
| **挑战** | 荒殒战场 | 区域 2 的尽头是历史残骸。区域 3 不再提供温柔过渡，敌人伤害和生命跳升让玩家必须认真判断是否推进。叙事上第一次暗示"这个世界有过大能，而大能也会陨落"。 | Lv.15 + atk ≥ 1,600 | "千年前就有人在这里死战。我得比他们更强，或者更聪明。" |

### 6.3 系统解锁与叙事节奏的匹配原则

| 原则 | 说明 | 结合 MVP 实例 |
|------|------|--------------|
| **一个系统 = 一个叙事理由** | 每个新系统解锁时，必须给出一个"为什么现在才出现"的世界观解释，而非凭空弹出菜单。 | 战斗解锁：灵气到达阈值 → "你感知到山中有异动"。离线结算：第一次退出返回 → "闭关归来，洞府中灵气自行汇聚"。 |
| **先行为，后界面** | 玩家先用最简方式体验系统核心行为，再获得完整 UI。 | 玩家先在没有区域选择界面的情况下默认在青丘山林战斗（行为），达到 Lv.5 后再让幽墟灵谷成为可选目标（界面）。 |
| **境界是仪式门禁，等级是 MVP 内容门禁** | 大境界突破用于制造强仪式感；区域推进在 Sprint 12 MVP 中采用等级/战力门槛，避免把金丹/元婴误写成已实现内容。 | 炼气 = 第一次境界跃迁。Lv.5 = Zone 2 内容门槛。Lv.15 = Zone 3 内容门槛。筑基 = Lv.30 MVP 终局仪式与 Phase 2 入口。 |
| **不过度解释，留白给想象** | 远古战场、大能残魂、剑中器灵——这些设定只需要散落在区域描述和掉落物品名中，不需要几百字 lore 面板。玩家自己拼凑比读设定集更有沉浸感。 | 荒殒战场不需要"前情提要"对话，只需要焦土上的剑痕和偶尔触发的幻音，外加几句战斗日志里的敌人名称暗示。 |
| **用限制制造渴望** | 玩家在区域 1 时，区域 2 的名称和推荐条件可见但状态为"锁定"。这种"我还没到但快了"的期望是推进的动力。 | 区域选择界面始终展示三个区域，锁定区域灰色显示，hover 提示具体等级/战力条件，例如"需 Lv.5 + atk ≥ 180"。 |

## 7. 叙事一致性与扩展约束

### 7.1 MVP 不破坏的世界观规则

- **玩家是散修，不归属任何宗门**。后续 Phase 5 引入宗门时，叙事衔接为"你的实力引起了 XX 宗门的注意，一位长老托人送来请帖……"而非"你一直在这个宗门里只是没做功能"。
- **境界不可回退**。突破后境界永久保留，不会出现"降级"叙事（数值上的软上限不计入叙事层）。
- **灵气是公有的自然能量，修为是个人的积累**。这一定义保证后续所有"灵气产出建筑""灵气浓度区域""灵气枯竭事件"等扩展不会产生语义矛盾。

### 7.2 为 225 系统保留的叙事锚点

| 未来系统 | 已在 MVP 中埋下的叙事种子 |
|----------|--------------------------|
| 宗门系统 (147-165) | 幽墟灵谷中有散修结庐——暗示这个区域已有非敌对修炼者，宗门离此不远。 |
| 法宝/器灵系统 (126, 138) | 荒殒战场古剑中的残存器灵——法宝有"灵性"的设定在 MVP 已建立。 |
| 天劫系统 (181, 192) | 元婴期描述提到"积蓄力量以突破"，暗示突破不是没有代价的。 |
| 飞升/世界层系统 (190, 193) | 境界命名体系预留至合道——每个境界名暗示更高层级的存在。 |
| 古修/秘境系统 (171) | 三个区域本质上都在探索"上古遗留"——青丘的禁制、灵谷的阵纹、战场的遗迹。秘境只是更大规模的上古遗留。 |
| 功法系统 (79) | 区域 2 和 3 的掉落表包含"残页"和"残卷"——功法作为可收集品的世界设定已建立。 |

### 7.3 文档版本控制与扩展流程

当后续阶段需要扩展世界观时：
1. 在本文对应节下追加新内容（如境界表新增"化神期"行），标注"Phase 2 新增"。
2. 保持术语表单一事实来源——新增术语追加到 §5.2 并注明阶段。
3. 如新区域在叙事上与前三个区域有空间重叠，重新审视 §3 的地理关系并更新区域过渡感设计表。
4. 重大世界观规则变更必须在 §7.1 中更新"不破坏规则"清单，并在全文中做矛盾扫描。
