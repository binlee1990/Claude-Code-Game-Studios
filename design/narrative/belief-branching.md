> 版本: v0.1 | 日期: 2026-04-26 | 状态: 草稿

## 1. 概述

本文档定义信念值三路线（仁/义/智）的完整分支矩阵：各章触发节点的位置、
阈值、权重、路线锁定时点，以及与各章 GDD 的引用关系。
信念值系统的设计目标是让玩家的每一次行动和选择"自然地"积累成立场，
而非在单一节点被强制二选一。路线分叉应在玩家事后回望时显得"理所当然"。

**设计原则（引自总纲）**：
- 信念值让分支"自然生长"而非"二选一"
- 选择 = 失去——每条路线有独立故事、角色和道德代价
- 3-5 个关键分支点控制总内容量
- 路线一旦锁定，后续章节的分支节点权重降低（强化已有路线，不易逆转）

---

## 2. 三路线定义

| 路线 | 属性键 | 核心立场 | 道德代价 | 独特资产（占位） |
|-----|-------|---------|---------|--------------|
| **仁路线** | `belief_values.ren` | 庇护弱者、以慈悲化干戈 | 有时纵容恶行，付出更大代价才能换来和平 | 流民势力盟友、王秀相关剧情线 |
| **义路线** | `belief_values.yi` | 维护秩序、以规则立威信 | 手段强硬，容易失去人心，盟友服从但不信任 | 豪强骑兵盟友、陈朗可能加入 |
| **智路线** | `belief_values.zhi` | 权衡利弊、以谋略制胜 | 被视为冷漠，两边都不完全信任 | 情报室升级解锁、隐藏地图事件 |

**属性值区间**：`[0, 100]`，每条路线独立计算，不互斥（可同时高仁高智）。

## 3. 分支节点矩阵

### 3.1 符号说明

- **节点 ID**：`B{章节}-N{序号}`，例如 B1-N1 = 第一章第一个节点
- **位置**：`战斗ID / 对话 / 开场选择` 三种触发类型
- **阈值**：仅在"路线分叉判断节点"（行 type = branch_gate）时有效
- **路线锁定**：`soft_lock`（后续节点权重减半）/ `hard_lock`（路线固定，其他路线节点不再触发）

### 3.5 路线锁定节点（全章概览）

| 锁定时点 | 锁定类型 | 触发条件 | 效果 |
|---------|---------|---------|------|
| Ch.3 结算后（B3-GATE） | soft_lock | 某路线领先 ≥ 20 分 | 后续节点该路线权重 ×1.5，其他路线 ×0.5 |
| Ch.4 中段（待定） | hard_lock | 某路线领先 ≥ 40 分，且差值不可逆 | 路线固定；进入对应路线专属章节内容；其他路线主线节点不再触发 |

**设计意图**：
- soft_lock 让玩家感到"我正在走某条路"，但仍可通过集中选择逆转（付出更大代价）。
- hard_lock 是"无法回头"的时刻——此后剧情分叉，内容量分裂为三条独立线。
- hard_lock 触发前有明确的叙事预警（NPC 台词暗示"你已经选了"）。

## 4. 阈值与权重规则

### 4.1 分叉判断公式（branch_gate 节点通用）

```
dominant = argmax(ren, yi, zhi)
margin = dominant_value - second_value

if margin >= BRANCH_THRESHOLD:
    branch = dominant_route
else:
    branch = default_route  # 见各章默认值
```

**各章 branch_gate 参数**：

| Gate ID | BRANCH_THRESHOLD | default_route | 说明 |
|---------|-----------------|---------------|------|
| B2-GATE | 5 | mercy（仁/智） | 义领先 ≥5 走 suppression；否则走 mercy |
| B3-GATE | 10 | 待定 | Ch.3 GDD 建立后填入 |
| soft_lock gate | 20 | 不锁定 | 任意路线领先 ≥20 触发 soft_lock |
| hard_lock gate | 40 | 不适用 | 任意路线领先 ≥40 触发 hard_lock |

> B2-GATE 的完整逻辑（含义路线专项判断）参见 design/gdd/chapter-02.md §3.8。

### 4.2 行为隐性累积权重（非选择型节点）

部分节点不弹出选择框，而是根据战斗行为自动计算信念值增减。权重基准如下：

| 行为类型 | 仁 | 义 | 智 |
|---------|---|---|---|
| 主动保护友方单位（护卫姿态触发） | +1/次，上限+5/战 | 0 | 0 |
| 使用控场技能（眩晕/定身）解围 | 0 | 0 | +1/次，上限+3/战 |
| 歼灭敌方精英单位（不含 Boss） | 0 | +2/个 | 0 |
| 己方单位退场（战场死亡） | -2/次 | 0 | 0 |
| 超额完成胜利条件（提前歼灭/零退场） | 0 | +3/战 | +2/战 |

> 行为累积权重是微调项，不是主要差值来源；主要差值由叙事选择节点驱动。

## 5. 路线锁定时点详述

### 5.1 Soft Lock（Ch.3 中段）

**触发条件**：
```
any(ren, yi, zhi) - max(other two) >= 20
```

**效果**：
- 后续所有节点中，领先路线权重 ×1.5（增幅放大）。
- 非领先路线权重 ×0.5（更难追回）。
- 叙事层：队友开始以对应路线立场称呼主角（例如义路线："将军，您一贯的铁腕......"）。
- 玩家仍可逆转：若连续选择非领先路线，累计差值可缩小至 20 以下，soft_lock 自动解除。

### 5.2 Hard Lock（Ch.4 中段）

**触发条件**：
```
any(ren, yi, zhi) - max(other two) >= 40
AND 当前章节已进入 Ch.4 以上
```

**效果**：
- 路线固定为领先路线，后续不再改变。
- 进入对应路线专属主线章节（内容量分裂为三条线）。
- 另外两条路线的对话选项和 NPC 不再出现（被替换为专属内容）。
- 存档记录 `locked_route = "ren" | "yi" | "zhi"`。

**叙事预警（hard_lock 触发前 1 章）**：
- 至少一个关键 NPC 有台词暗示"你已经选了"。
- 叙事总监负责编写具体台词；本文档只定义触发时机（Ch.4 开章前的章节结算屏）。
